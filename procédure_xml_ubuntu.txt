BEFORE :

Création 2 dossiers : -----"Faire des 'cd' pour arriver au bon fichier "-----
>>> mkdir -p images/positif (qq images de l'objet dans un environnement) -> ex:photo avec panneau stop
>>> mkdir -p images/négatif (bcp d'images sans le truc qu'on cherche) -> ex:sans panneau stop

Mettre ses images ou les télécharger -> même taille : " programme pour classer dans les dossiers et les redimensionner"

Création du fichier negatif.txt ou "background": 
>>>ls *.jpg > negatif.txt (écrit toutes les images négatives dans le 'txt' avec des numméros)

Génération des images positives : -----"Avoir une image simple de l'objet à detecter : 'stop.jpg"-----
On utilise l'outil 'opencv_createsamples'

>>> opencv_createsamples -img img_stop.jpg -bg negatives -info positif/positif.lst -pngoutput -maxxangles 0.5 -maxyangle 0.5 -maxzangle 0.5 -num 400
>>> opencv_createsamples -info positif/positif.lst -num 181 -w 20 -h 20 -vec positives.vec  (fichier binaire)

AFTER : 

>>> mkdir data
>>> opencv_traincascade -data data -vec positif.vec -bg list.txt, -numPos 181 - numNeg 181 -numStages 10 -w 20 -h 20
